# TinyUSB
Forked from TinyUSB v0.18.0 (https://github.com/hathach/tinyusb)

The repository was stripped down to only include the needed files. Only the `src` folder is included

To update the repo with upstream, clone a local version of the upstream tinyusb repo. Use a diff tool such as meld to compare this folder with the upstream folder. Only bring back useful files or changes.
